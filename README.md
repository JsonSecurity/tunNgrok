# tunNgrok
This repository serves to automate The installation of NGROK in Termux - 2024

<h3>PLATAFORMAS</h3>
- Termux

<h3>INSTALACION</h3>
<pre>git clone</pre>
git clone
chmod +x ./install.sh
./install.sh
```
<h3>Config (PATH)</h3>
```
./ngrok.sh
```
Restart your terminal


<h3>Help</h3>
```
tun
```
<h3>Tunel TCP</h3>
```
tun -s tcp -p 443
```
<h3>Tunel HTTP</h3>
```
tun -s http -p 8080
```
