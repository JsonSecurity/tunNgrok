# tunNgrok
This repository serves to automate The installation of NGROK in Termux - 2024

<h3>PLATAFORMAS</h3>
- Termux

##INSTALACION
  ```
git clone
chmod +x ./install.sh
./install.sh
```
#Config (PATH)
```
./ngrok.sh
```
Restart your terminal

#Help
```
tun
```
#Usage (Tunels)
```
tun -s tcp -p 443
```
