# tunNgrok
This repository serves to automate The installation of NGROK in Termux - 2024

<h3>PLATAFORMAS</h3>
- Termux

<h3>INSTALACION</h3>
<pre>
  git clone
  git clone
  chmod +x ./install.sh
  ./install.sh
</pre>


<h3>Config (PATH)</h3>
<pre>
  ./ngrok.sh
</pre>
Restart your terminal

<h3>Help</h3>
<pre>
  tun
</pre>

<h3>Tunel TCP</h3>
<pre>
  tun -s tcp -p 443
</pre>

<h3>Tunel HTTP</h3>

<pre>
 tun -s http -p 8080
</pre>
